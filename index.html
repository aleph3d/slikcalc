Slikcalc

Slikcalc is a jQuery plugin that makes it easy to link a mathematical formula to a group of text inputs, and have them calculate a value.

  • A formula is written plainly, wrapping variables (inputs) with curly brackets. The formula, "a * b + c" would translate into "{a} * {b} + {c}".
  • Those variables are then defined in a config object passed into the slikcalc() function. The id of the variable's input field must be defined, along with any other optional parameters desired. The variable "a" would be defined as "a : {elementId : 'ex-1-1'}"
