Slikcalc

Slikcalc is a small javascript framework that makes it easy to add dynamic and static mathematical calculations to your html pages. Slikcalc is built to work in conjunction with any of the big javascript libraries out there. The current libraries supported are:

  • yui
  • jQuery
  • dojo
  • mootools
  • prototype
* if a framework you use is not currently supported, its easy to create an adapter to add support, just look at the provided adapters and follow their patterns)

Setup

Setting up Slikcalc is super simple, just download the archive, and then include your javascript library of choice, slikcalc.js, along with the library-specific adapter. After that, you're ready to begin with either a ColumnCalc or FormulaCalc, or if you're feeling really smart, a Custom calculator.

Column Calculations

  • Description
  • Options
  • Examples
Column calculators are for when you have a collection of values that you want to apply a mathematical computation to, such as summing them, or multiplying them together. Usage includes creating a new ColumnCalc object, which can take configuration options, and then call the addRow() method for each value you want to add, which can also take configuration options.
Constructor options:
  • : (Required) Element ID to place end result of column calculation
  • : (Optional) ( +, -, *, x, / ) Mathematical operator to apply against each column to produce end result. Defaults to '+'
  • : (Optional) true/false(default) If true, event listeners are attached to inputs that fire the calculate method
  • : (Optional) true/false(default) If true, on page load the calculate method is fired.
addRow() options:
  • : Element id that holds the value to calculate
  • : Checkbox object that defines the behavior of a checkbox
  • : Element id of checkbox. Required if 'checkbox' included
  • : true/false(default) If true, row is included in total calculcation when un-checked, and omitted when checked.

Addition Column Calc

var columnCalc1 = new slikcalc.ColumnCalc({
	totalId: 'cc-1-total',
	registerListeners: true,
	calcOnLoad: true
});
columnCalc1.addRow({ id: 'cc-1-1' });
columnCalc1.addRow({ id: 'cc-1-2' });
columnCalc1.addRow({
	id: 'cc-1-3',
	checkbox: { id: 'cc-1-3-c' } 
});

  • A formula is written plainly, wrapping variables (inputs) with curly brackets. The formula, "a * b + c" would translate into "{a} * {b} + {c}".
  • Those variables are then defined in a config object passed into the slikcalc() function. The id of the variable's input field must be defined, along with any other optional parameters desired. The variable "a" would be defined as "a : {elementId : 'ex-1-1'}"
